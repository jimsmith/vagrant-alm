# vagrant-alm
Provisioning ALM environment using Vagrant

Softwares:

Gitblit : Source Control Management (SCM)
Jenkins: Continuous Integration (CI)
Sonar: software quality QA
Nexus: ...
Redmine: Missing...

Jenkins: 8085 port.
Sonar: 8086 port.
Gitblit: 8087 (HTTP) 8088 (HTTPS) ports.
Nexus: 8089 port, /nexus path
